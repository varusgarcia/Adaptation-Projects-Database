Adaptation Projects Database

Visualisation of the Ci:Grasp adaptation project database

Project by Alvaro Garcia

Created for the course Information Visualization by Marian Dörk at Fh Potsdam

With this project I wanted to create an interactive visualisation of the adaptation projects collected on the PIC Database.
The idea was to visualise the impacts, sector and stimuli for each project and to display that information on a map.
I decided that the three elements defining each project would be color coded, generating an icon for each one of the projects.
These three elements can be found respectively on the left, bottom and right of the map showing each the amount of projects displayed. By clicking on one of them, the projects that don’t have that element are hidden, and we can see how the amount changes.
In this way we can see the relation between the three elements.

Data provided by the Potsdam Institute for Climate Impact Research
